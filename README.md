# Linux Student

## Working with Linux Shell
- ```echo "Hello"``` Print any string
- ```uptime```  Linux user and active status

## Type of command in linux
- Internal Or Built-in commands. EX:  ``` echo, cd, pwd```
- External Commands. Ex: ``` mv, date, uptime ```

## Shell Types
- Bourne Shell
- C Shell
- Korn Shell
- Z Shell
- Bourne again Shell
